27/*
28 * Functions: CARDINAL32 CalculateCRC
29 * void Build_CRC_Table
30 *
31 * Description: The functions in this module provide a means of calculating
32 * the 32 bit CRC for a block of data. Build_CRC_Table must
33 * be called to initialize this module. CalculateCRC must
34 * NOT be used until after Build_CRC_Table has been called.
35 * Once Build_CRC_Table has been called, CalculateCRC can
36 * be used to calculate the CRC of the data residing in a
37 * user specified buffer.
38 *
39 * Notes: Build_CRC_Table only needs to be called once. Once the internal
40 * CRC table has been built, there is no need to build it again.
41 *
42 */

61/* The following define is used as the initial CRC value passed to CalculateCRC whenever a new CRC calculation is begun.
62 For example, if you wish to calculate the CRC for an object which is 12 K, and your buffer is 2 K, you would enter
63 a loop which reads in 2K of the object, and calculates the CRC for the buffer. On each call to CalculateCRC, the
64 CRC value passed in would be the one calculated on the previous iteration of the loop. But, for the first iteration
65 of the loop, there is no previous value. In this case, you use INITIAL_CRC as the previous value. */
66#define INITIAL_CRC 0xFFFFFFFFL
