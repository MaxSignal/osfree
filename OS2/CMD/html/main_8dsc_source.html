71/*! @page page3 Command-Line Editing
72
73The command line works like a single-line word processor, allowing you to
74edit any part of the command at any time before you press :hp2.Enter:ehp2.
75to execute it, or :hp2.Esc:ehp2. to erase it. The command line you enter
76can be up to 1023 characters long.
77:p.
78You can use the following editing keys when you are typing a command (the
79words :hp2.Ctrl:ehp2. and :hp2.Shift:ehp2. mean to press the Ctrl or Shift
80key together with the other key named)&colon.
81:p.
82:hp2.Cursor Movement Keys&colon.:ehp2.
83:lm margin=8.:parml compact tsize=16 break=fit.
84:pt.:hp2.␛:ehp2.:pd.Move the cursor left one character.
85:pt.:hp2.␚:ehp2.:pd.Move the cursor right one character.
86:pt.:hp2.Ctrl ␛:ehp2.:pd.Move the cursor left one word.
87:pt.:hp2.Ctrl ␚:ehp2.:pd.Move the cursor right one word.
88:pt.:hp2.Home:ehp2.:pd.Move the cursor to the beginning of the line.
89:pt.:hp2.End:ehp2.:pd.Move the cursor to the end of the line.
90:eparml.:lm margin=1.
91.br
92:hp2.Insert and Delete&colon.:ehp2.
93:lm margin=8.:parml compact tsize=16 break=fit.
94:pt.:hp2.Ins:ehp2.:pd.Toggle between insert and overtype mode.
95:pt.:hp2.Del:ehp2.:pd.Delete the character at the cursor.
96:pt.:hp2.Backspace:ehp2.:pd.Delete the character to the left of the cursor.
97:pt.:hp2.Ctrl-L:ehp2.:pd.Delete the word or partial word to the left of the cursor.
98:pt.:hp2.Ctrl-R:ehp2.:pd.or :hp2.Ctrl-Bksp:ehp2. Delete the word or partial word to the right of the cursor.
99:pt.:hp2.Ctrl-Home:ehp2.:pd.Delete from the beginning of the line to the cursor.
100:pt.:hp2.Ctrl-End:ehp2.:pd.Delete from the cursor to the end of the line.
101:pt.:hp2.Esc:ehp2.:pd.Delete the entire line.
102:eparml.:lm margin=1.
103.br
104:hp2.Execution&colon.:ehp2.
105:lm margin=8.:parml compact tsize=16 break=fit.
106:pt.:hp2.Ctrl-C:ehp2.:pd.or :hp2.Ctrl-Break:ehp2. Cancel the command line.
107:pt.:hp2.Enter:ehp2.:pd.Execute the command line.
108:eparml.:lm margin=1.
109.br
110Most of the command-line editing capabilities are also available
111when a &4OS2. command prompts you for a line of input. For example,
112you can use the command-line editing keys when
113:link reftype=hd refid=DESCRIBE.DESCRIBE:elink. prompts for a file
114description, when :link reftype=hd refid=INPUT.INPUT:elink. prompts for
115input from an alias or batch file, or when :link reftype=hd
116refid=LIST.LIST:elink. prompts you for a search string.
117:p.
118If you want your input at the command line to be in a different
119color from &4OS2.'s prompts or output, you can use the Display page of the
120:link reftype=hd refid=OPTION.OPTION:elink. dialogs, or the
121:link reftype=hd refid=INI_InputColors.InputColors:elink. directive in
122:hp1.&4OS2INI.:ehp1..
123:p.
124&4OS2. will prompt for additional command-line text when you include
125the :link reftype=hd refid=Escape.escape character:elink. as the very
126last character of a typed command line. The default escape character
127is the caret [:hp2.^:ehp2.]. For example&colon.
128:xmp.
129 [c&colon.\] echo The quick brown fox jumped over the lazy^
130 More? sleeping dog. > alphabet
131:exmp.:p.
132Sometimes you may want to enter one of the command line editing keystrokes
133on the command line, instead of performing the key's usual action. For
134example, suppose you have a program that requires a Ctrl-R
135character on its command line. Normally you couldn't type this
136keystroke at the prompt, because it would be interpreted as a
137"Delete word right" command.
138:p.
139To get around this problem, use the special keystroke :hp2.Alt-255:ehp2..
140You enter Alt-255 by holding down the :hp2.Alt:ehp2. key while you type
141:hp2.255:ehp2. on the numeric keypad, then releasing the :hp2.Alt:ehp2. key (you
142must use the number keys on the numeric pad; the row of keys at the
143top of your keyboard won't work). This forces &4OS2. to interpret
144the next keystroke literally and places it on the command line,
145ignoring any special meaning it would normally have as a command-line
146editing or history keystroke. You can use Alt-255 to suppress
147the normal meaning of command-line editing keystrokes even if they
148have been reassigned with :link reftype=hd refid=INIKeyMap.key mapping directives:elink. in
149the :hp1..INI:ehp1. file, and Alt-255 itself can be reassigned with the
150:link reftype=hd refid=INI_CommandEscape.CommandEscape:elink. directive.
