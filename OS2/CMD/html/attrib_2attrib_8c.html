sets flags and/or options field based on characters found in param string, and action_type specifier

Sets program otions or attribute action codes, based on parameter string, and action type passed in. Parameter string, is being examined only for len characters.

Parameters
flagspointer to an ATTRIB structure, where recognized attribute action codes should be stored
paramparameter string
lenlength of valid parameter characters. Characters above len'th are ignored
action_typeaction-type character, can be one of following: PARAM_EQUAL - '=', which means setting specified by param attributes, and removing rest PARAM_SET - '+', which means setting on, specified attributes PARAM_CHAR1 - '/', which means that, param string contains an option selector PARAM_CHAR2 - '-', which means removing, specified attributes PARAM_NEG - '~', which means negating, specified attributes
optionspointer to an OPTIONS structure, where recognized options settings should be stored
Returns
NO_ERROR - successfull completition 1 - parameter contained invalid character (could not be recognized)
